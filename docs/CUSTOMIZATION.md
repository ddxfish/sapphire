
## Full Customizations for YOUR personas

Each can have a ton of variations and you can switch between them instantly by switching chats.

### Make the settings yours
- Gear icon > App Settings 
- Change names and avatars to yours
- Enable TTS and STT, Wakeword
- Pick your wakeword
- (wakeword) adjust background noise setting

### Make the Prompt yours
- Prompt editor in the sidebar , then click + 
- Choose Assembled (more customizable) and name it what you want
- Click the + next to Persona, name YOUR persona and who they are 
- Click the + next to Relationship, define who you are to the AI
- (optional) Add Location, (AI) Goals, Text Format, Scenario, Extras (multiple), Emotions (multiple)
- SAVE the prompt with the disk icon

### Make the default chat yours
- Open default chat (upper left). Click ... > Chat Settings
- select your preferred default prompt you set up
- select what tools if any you want the AI to have
- TTS voice, pitch, speed
- SPICE adds randomness to each reply
- Inject Date the AI knows the date
- Custom text it ALWAYS knows in addition to system prompt
- SET AS DEFAULT then save. 

### Make a second persona
- Click ... next to chat name > New chat
- next to new chat name > ... > Chat Settings
- Same as the last section, but dont save as default, just save
