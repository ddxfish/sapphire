Quick Start

# Linux only
sudo apt-get install libportaudio2

# Create environment
conda create -n sapphire python=3.11
conda activate sapphire

# Clone and run
git clone https://github.com/ddxfish/sapphire.git
cd sapphire
pip install -r requirements.txt
python main.py

# Web UI at https://localhost:8073

Requires Python 3.11+, 16GB+ RAM. Nvidia GPU recommended for TTS/STT. Local LLM via LM Studio or cloud via Claude/OpenAI.
